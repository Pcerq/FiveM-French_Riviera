# FiveM-French_Riviera
## GTA IV's French Riviera map ported to fiveM

### *Features*:
* Proper V shaders for the map.
* Updated textures and better UV.
* Proper normal and spec maps alongside detail maps.
* V props added alongside IV props.
* LOD & SLOD support.
* Added V Traffic path with pedestrians
